# Circus求人媒体サイト自動登録システム 設定手順書

## 1. 概要
企業マスタから未登録企業を自動的にCircus求人媒体サイトに登録し、求人票を作成・公開する自動化システムです。デスクトップ上のHTMLファイルをベースとしたデモ環境での動作を想定しています。

## 2. 事前準備

### 2.1 必要な環境
- Windows 10/11
- Google Chrome（最新版）
- Robo-Pat DX（最新版）
- Microsoft Excel（企業マスタファイル読み込み用）
- デスクトップへの読み書き権限

### 2.2 対象システムの設定
- デスクトップに以下のHTMLファイルを配置
  - `求人登録.html`
  - `circus_company_demo.html`
- 企業マスタExcelファイルをデスクトップに配置
- Chromeブラウザの既定プロファイル設定
- メール送信機能の設定（SMTP設定）

## 4. スクリプトの実行手順

### 4.1 初回実行前の確認事項
- [ ] デスクトップにHTMLファイルが正しく配置されている
- [ ] 企業マスタExcelファイルが準備されている
- [ ] Chromeブラウザが正常に起動する
- [ ] 企業マスタに必要な項目（企業名、住所、電話番号等）が含まれている
- [ ] メール送信設定が完了している
- [ ] ネットワーク接続が安定している

### 4.2 実行方法
1. **Robo-Pat DXを起動**
   - システムトレイからRobo-Pat DXを起動
   - 作成したスクリプトを選択

2. **変数値の確認・設定**
   - 実行前に変数設定画面で各項目の値を確認
   - 必要に応じて企業情報、求人情報を更新

3. **スクリプト実行**
   - 「実行」ボタンをクリック
   - 処理状況をログウィンドウで確認

4. **実行中の監視**
   - 各企業の登録処理進捗を確認
   - エラーが発生した場合は自動的に次の企業へスキップ

5. **完了確認**
   - 処理完了後、通知メールを確認
   - ログファイルで詳細な実行結果を確認

## 5. トラブルシューティング

### よくある問題と対処法
| 症状 | 原因 | 対処法 |
|------|------|--------|
| HTMLファイルが開けない | ファイルパスが間違っている | デスクトップにファイルが存在するか確認、パス設定を見直し |
| 企業情報が入力されない | セレクターが間違っている | HTMLファイルの要素IDを確認、セレクター設定を修正 |
| 求人公開ボタンが押せない | 必須項目が未入力 | 全ての必須フィールドに値が設定されているか確認 |
| メール送信に失敗 | SMTP設定が未完了 | メール送信設定を再確認、認証情報を確認 |
| Chromeが起動しない | Chrome実行ファイルパスエラー | Chrome再インストール、パス設定確認 |
| 処理が途中で止まる | 画面遷移待機時間不足 | wait_for_screen_calms の待機時間を延長 |
| 業種選択に失敗 | 選択肢の値が一致しない | HTMLファイルのoption値を確認、変数値を調整 |

## 6. 注意事項
- **実行環境**: デモ用HTMLファイルでの動作のため、本番環境では画面構成が異なる可能性があります
- **データバックアップ**: 企業マスタファイルは実行前にバックアップを取得してください
- **エラーハンドリング**: エラーが発生した企業は自動的にスキップされますが、ログを確認して手動対応が必要な場合があります
- **実行タイミング**: 大量データ処理時は、サーバー負荷を考慮して実行時間を調整してください
- **権限確認**: ファイル読み書き、ブラウザ操作に必要な権限が付与されているか確認してください
- **メール設定**: 通知メール機能を使用する場合は、事前にSMTP設定を完了させてください
