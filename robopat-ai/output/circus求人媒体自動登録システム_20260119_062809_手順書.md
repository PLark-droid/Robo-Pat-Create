# circus求人媒体自動登録システム 設定手順書

## 1. 概要
circusデモサイトにて企業情報の自動登録および求人票の作成・公開を行う自動化処理。未登録企業の企業マスタからの登録と求人票作成を一連の流れで自動化します。

## 2. 事前準備

### 2.1 必要な環境
- Windows OS
- Google Chrome ブラウザ
- Microsoft Excel（企業マスタファイル用）
- Robo-Pat DX

### 2.2 対象システムの設定
- デスクトップに `circus_company_demo.html` と `求人登録.html` を配置
- `企業マスタ.xlsx` ファイルをデスクトップに配置
- 企業マスタファイルは以下の列構成で作成：
  - A列：企業名
  - B列：住所
  - C列：電話番号
  - D列：メールアドレス
  - E列：求人職種
  - F列：職種説明
  - G列：給与情報
  - H列：雇用形態
  - I列：勤務地
  - J列：勤務時間
  - K列：応募条件

## 3. 変数の設定

| 変数名 | 説明 | 設定値の例 |
|--------|------|-----------|
| current_company | 処理対象の企業名 | サンプル株式会社 |
| company_address | 企業住所 | 東京都港区○○1-1-1 |
| company_phone | 企業電話番号 | 03-1234-5678 |
| company_email | 企業メールアドレス | info@sample.co.jp |
| job_title | 求人職種名 | システムエンジニア |
| job_description | 職種詳細説明 | システム開発・保守業務 |
| salary_range | 給与情報 | 年収400-600万円 |
| employment_type | 雇用形態 | 正社員 |
| work_location | 勤務地 | 東京都内 |
| work_hours | 勤務時間 | 9:00-18:00 |
| job_requirements | 応募条件 | IT業界経験2年以上 |
| search_result | 企業検索結果 | （システム取得） |
| publish_result | 求人公開結果メッセージ | （システム取得） |
| error_message | エラーメッセージ格納用 | （システム取得） |
| job_error_message | 求人登録エラーメッセージ | （システム取得） |

## 4. スクリプトの実行手順

### 4.1 初回実行前の確認事項
- [ ] デスクトップに必要なHTMLファイルが配置されている
- [ ] 企業マスタExcelファイルが正しい形式で準備されている
- [ ] Google Chromeが正常に動作する
- [ ] 変数設定が完了している
- [ ] メール通知設定が完了している（エラー通知用）

### 4.2 実行方法
1. Robo-Pat DXを起動
2. 「circus求人媒体自動登録システム」スクリプトを開く
3. 変数値を確認・調整
4. スクリプトを実行
5. 実行ログを確認して正常完了を確認

## 5. トラブルシューティング

### よくある問題と対処法
| 症状 | 原因 | 対処法 |
|------|------|--------|
| HTMLファイルが開かない | ファイルパスが間違っている | デスクトップにファイルが配置されているか確認 |
| 企業検索結果が取得できない | セレクタが正しくない | HTMLファイルの要素IDを確認 |
| 求人公開に失敗する | 必須項目が未入力 | 変数設定値を確認し、全ての必須項目に値が設定されているか確認 |
| エラーメールが送信されない | メール設定が未完了 | メール送信設定を確認し、正しいメールアドレスが設定されているか確認 |
| スクリプトが途中で停止する | 画面読み込みが完了していない | wait_for_screen_calmsの待機時間を延長 |
| ドロップダウンの選択に失敗 | 企業名が登録されていない | 企業登録処理が正常完了しているか確認 |

## 6. 注意事項
- 本システムはデモサイト用のため、実際の求人媒体では要素IDやセレクタが異なる場合があります
- 企業マスタファイルの形式を変更する場合は、スクリプト内の変数設定も合わせて変更してください
- エラーが発生した場合は、スクリーンショットが自動保存されるため、ログと合わせて確認してください
- 大量データを処理する場合は、途中でブラウザのメモリ不足が発生する可能性があります
- 実行中はPCの操作を控えてください（マウスやキーボード操作によりスクリプトが中断される可能性があります）
